# Описание проекта

Проект представляет собой статическое веб-приложение для визуализации пользовательских измерений роста, веса и окружности головы на референсном графике 0-48 месяцев.

## Архитектура

Страница состоит из трёх слоёв:

- PNG-фон с исходным графиком;
- SVG-слой с пользовательскими линиями и точками;
- HTML-слой с таблицей, предупреждениями и кнопками управления.

Калибровка шкал вынесена в `js/chart-calibration.js`. Для роста, веса и окружности головы используются отдельные функции пересчёта значений в координату Y, чтобы изменение одной шкалы не влияло на остальные.

Пользовательские данные обрабатываются в `js/user-data.js`. Значения проходят проверку диапазонов, сохраняются в `localStorage` и могут быть полностью удалены кнопкой «Очистить данные».

Печать реализована через `window.print()` в `js/export-print.js`. Печатное представление задаётся отдельным файлом `css/print.css`.

## Валидация

Рекомендуемые диапазоны ввода:

- возраст: 0-48 месяцев;
- рост: 30-120 см;
- вес: 1-30 кг;
- окружность головы: 25-70 см.

Если значение находится вне диапазона графика, рядом с ячейкой показывается предупреждение. Остальные корректные точки продолжают отображаться.

## Конфиденциальность

Приложение не использует сервер и не отправляет данные наружу. Данные MVP сохраняются только на устройстве пользователя в `localStorage`.

## Дисклеймер

Инструмент предназначен для визуализации данных и не заменяет консультацию врача. Интерпретацию графиков должен выполнять медицинский специалист.
