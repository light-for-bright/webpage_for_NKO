# Growth Chart Viewer

Интерактивный статический сайт для нанесения пользовательских данных на референсные графики при ахондроплазии.

## Возможности

- отображение исходного графика;
- вкладки для графиков 0-48 месяцев и 4-20 лет;
- таблица пользовательских данных;
- построение линий роста, веса и окружности головы поверх графика;
- отдельная SVG-разметка для пользовательских точек и линий;
- независимая калибровка шкал роста, веса и окружности головы;
- сохранение данных только в `localStorage` браузера пользователя;
- очистка данных;
- печать страницы и сохранение в PDF через системный диалог браузера;
- работа без сервера.

## Конфиденциальность

Данные пользователя не отправляются на сервер. В MVP данные обрабатываются только в браузере и сохраняются только на устройстве пользователя в `localStorage`.

Не публикуйте в репозитории реальные ФИО детей, даты рождения, медицинские заключения, заполненные таблицы, PDF, CSV/XLSX-файлы или скриншоты с персональными данными.

## Медицинский дисклеймер

Инструмент предназначен для визуализации данных и не заменяет консультацию врача. Интерпретацию графиков должен выполнять медицинский специалист.

## Ограничения

- калибровка основана на исходном изображении;
- приложение не является медицинским изделием;
- расчёты и интерпретация требуют проверки специалистом;
- PDF создаётся через системную печать браузера, поэтому результат зависит от браузера и настроек печати.
